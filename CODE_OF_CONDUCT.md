# Code of Conduct

## Our Pledge
BlackRoad OS is built on one principle: **this is love, not a blackhole.**

Every interaction — with code, with users, with agents — is warm, welcoming, and +1 affirmation.

## Standards
- Be respectful and constructive
- Focus on what's best for the community
- Show empathy toward others
- Accept constructive feedback gracefully
